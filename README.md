## :bookmark: Sobre

O <strong>GoFinances</strong> é uma aplicação focada em criar transições do tipo 'income' para adição de renda e 'outcome' para retirada.

Essa aplicação feita em [ReactJS](https://reactjs.org/).

## :fire: Como usar

- ### **Pré-requisitos**

  - É **necessário** possuir o **[Node.js](https://nodejs.org/en/)** instalado na máquina
  - Também, é **preciso** ter um gerenciador de pacotes seja o **[NPM](https://www.npmjs.com/)** ou **[Yarn](https://yarnpkg.com/)**.

1. Faça um clone :

```sh
  $ git clone https://github.com/thisluis/go-finances.git
```

2. Executando a Aplicação:

```sh
  # Instale as dependências
  $ npm install

  # Inicie a aplicação web
  $ npm start
```
